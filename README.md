# PolyPulse

A real-time Polymarket analytics dashboard built with Next.js and Tailwind CSS.

## Features
- 🔥 Trending markets by 24h volume
- ⚡ Biggest probability movers
- 🕒 Recent market activity feed

## Data Source
Uses the public Polymarket Gamma API — no API key required.
`https://gamma-api.polymarket.com`

## Getting Started

npm install
npm run dev

Open http://localhost:3000

## Deploy
One-click deploy on Vercel. No environment variables needed.

## Built with
- Next.js 15
- Tailwind CSS
- Polymarket Gamma API
